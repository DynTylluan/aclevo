SuperNova Webserver Teklik Development

Project Description

SuperNova WebServer is a simple webserver coded in Python without the hassle of messing around in configuration files. It allows you to host static content easily, with just one simple command as opposed to other platforms which require much more configuration to use. It can even be ran as a user without administrator permission (some features may be restricted). The port can be easily changed without having to go through configuration files.
