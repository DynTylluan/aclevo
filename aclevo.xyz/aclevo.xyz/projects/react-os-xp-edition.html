React OS: XP Edition Teklik Development

Project Description

React OS: XP Edition is an operating system redesign made by ex-Aclevo Member YTA during his time in Deleno. Based on React OS 0.4.6, it brings in some graphics from Windows XP and applies them to the Free Operating System.There are currently two versions of the ISOs available for use: The Live Version and the Installable Version. Please note that these versions may not be swapped around for the opposite use, as doing so may cause a bugcheck (BSOD).
